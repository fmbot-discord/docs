# ⚠️ Spotify connection failed

Your Spotify account wasn't connected. This usually happens when:

- You declined the authorization on Spotify's page
- The connection link expired — links are only valid for a few minutes
- You opened an old or already-used link

To try again, go to Discord, run `/remote` (or `.remote`), and click **Connect Spotify** to try again.

- [Support server](https://discord.gg/fmbot)
